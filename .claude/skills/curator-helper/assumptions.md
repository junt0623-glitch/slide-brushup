# assumptions.md — curator-helper

ユーザーから明示されなかったため、以下を既定値として採用した。

| 項目 | 採用した既定値 | 理由 |
|---|---|---|
| 出力言語 | 依頼文の言語に合わせる(基本は日本語、英語版が必要な場合は明示的に依頼) | 学芸員の勤務先が日本国内の美術館であるため |
| キャプションの長さ | 指定がなければ日本語200字前後(館内解説の一般的な目安) | 業界的に多い目安値。実際の館の規定があれば必ずそちらを優先する |
| 専門分野の前提知識 | 依頼者は中国陶磁を含む美術史の専門知識を持つ前提で会話するが、来館者向け文章では平易に言い換える | ユーザーが中国陶磁専門の学芸員であるため |
| テスト運用範囲 | Claude Code上での動作確認を基本とし、claude.ai(ブラウザ版)でも同じSKILL.mdが使えることを想定 | ユーザー環境がClaude Codeであるため |
| Web検索の使用 | 事実確認が必要な場合のみ、確認済みの検索ツールを使う。使えない環境では「要確認」と明記する | 環境によりツール有無が異なるため |

これらは初期値であり、実際の運用で館の規定(文字数、文体、承認フロー等)が分かり次第、SKILL.mdまたはreferences/を更新して反映する。
