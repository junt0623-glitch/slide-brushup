# quality-rubric.md

いつ読むか: 成果物を出す直前の自己チェック時。
何を判断するために読むか: 「AIっぽい」「テンプレっぽい」文章になっていないかを判定するため。

## 減点対象(見つけたら書き直す)
- 「〜という魅力があります」「必見です」「ぜひご覧ください」で締める紋切り型の結び
- 抽象的な形容詞の連発(「素晴らしい」「美しい」を具体的描写なしに使う)
- 箇条書きの多用による、思考停止した情報の羅列(文章として読ませる箇所は文章で書く)
- 同じ文末表現(「〜です。」「〜ます。」)が3文以上連続する
- 存在確認できない年代・人物名・文献名の断定

## 加点対象
- 視覚的・具体的な描写(色、質感、造形の特徴を名詞ではなく描写で伝える)
- 事実と解釈が区別されている
- 読み手(来館者、審査員、記者など)が誰かを意識した語彙選択になっている
- 一文の長さに強弱がある(単調でない)

## 自己チェックの手順
1. 結びの一文を確認し、紋切り型なら書き直す
2. 抽象的な形容詞をハイライトし、具体描写に置き換えられないか検討する
3. 文末表現の連続をチェックする
4. 断定している事実を1つずつ、入力情報から追跡できるか確認する
